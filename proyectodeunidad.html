Investigación de Etiqueta HTML


1.Nombre: html

● Inicio: < html>
● Cierre: < /html>
● Descripción: Es como el contenedor general de toda tu página web. Todo lo que hagas tiene que estar metido aquí, sí o sí.

2.Nombre: head

● Inicio: < head>
● Cierre: < /head>
● Descripción: Parte oculta del sitio. Aquí va lo que no se ve en pantalla pero es clave, como el título del sitio o links a archivos externos.

3.Nombre: title

● Inicio: < title>
● Cierre: < /title>
● Descripción: Define el título que aparece en la pestaña del navegador. Como cuando entras a YouTube y dice "YouTube" arriba.

4.Nombre: body

● Inicio: < body>
● Cierre: < /body>
● Descripción: Todo lo que el usuario ve en la página (textos, imágenes, etc.) va dentro de aquí.

5.Nombre: h1 hasta h6

● Inicio: < h1> (y así hasta < h6>)
● Cierre: < /h1> (y así hasta )
● Descripción: Son los títulos. h1 es el más grande y h6 el más chico. Úsalos para jerarquizar el contenido.

6.Nombre: p

● Inicio: < p>
● Cierre: < /p>
● Descripción: Crea párrafos. Ideal para escribir bloques de texto normales.

7.Nombre: br

● Inicio: < br>
● Cierre: No tiene cierre
● Descripción: Salto de línea. Como presionar “Enter” para cambiar de renglón.

8. Nombre: hr

● Inicio: < hr>
● Cierre: No tiene cierre
● Descripción: Línea horizontal que divide contenido. Como una raya para separar secciones.

9. Nombre: a

● Inicio: < a>
● Cierre: < /a>
● Descripción: Crea enlaces. Se usa con href para mandar al usuario a otra página o lugar.

10. Nombre: img

● Inicio: < img>
● Cierre: No tiene cierre
● Descripción: Inserta imágenes. Le tenés que poner src para decir qué imagen usar.

11. Nombre: strong

● Inicio: < strong>
● Cierre:
● Descripción: Pone el texto en negrita y lo marca como importante.

12. Nombre: em

● Inicio: < em>
● Cierre: /em>
● Descripción: Hace el texto cursiva y lo destaca como algo a enfatizar.

13. Nombre: ul

● Inicio: < ul>
● Cierre: < /ul>
● Descripción: Lista sin orden (con viñetas o puntos). Como cuando hacés una lista de cosas a comprar.

14. Nombre: ol

● Inicio: < ol>
● Cierre: < /ol>
● Descripción: Lista ordenada, con números. Ideal para pasos o rankings.

15. Nombre: li

● Inicio: < li>
● Cierre: < /li>
● Descripción: Cada elemento de una lista, ya sea con viñetas (ul) o números (ol).

16. Nombre: div

● Inicio: < div>
● Cierre: < /div>
● Descripción: Caja invisible que agrupa cosas. Súper útil para organizar contenido y diseño.

17. Nombre: span

● Inicio: < span>
● Cierre: < /span>
● Descripción: Sirve para aplicar estilo a un pedazo de texto sin afectar todo el bloque.

18. Nombre: input

● Inicio: < input>
● Cierre: No tiene cierre
● Descripción: Campo para que el usuario escriba o elija algo. Puede ser texto, checkbox, etc.

19. Nombre: form

● Inicio: < form>
● Cierre: < /form>
● Descripción: Agrupa campos para enviar información. Como cuando llenás un formulario para registrarte.

20. Nombre: label

● Inicio: < label>
● Cierre: < /label>
● Descripción: Es el texto que acompaña a los campos. Te dice qué deberías escribir ahí.

21. Nombre: button

● Inicio: < button>
● Cierre: < /button>
● Descripción: Botón que el usuario puede presionar. Puede enviar un formulario o hacer otra acción.

22. Nombre: select

● Inicio: < select>
● Cierre: < /select>
● Descripción: Menú desplegable para elegir una opción entre varias.

23. Nombre: option

● Inicio: < option>
● Cierre: < /option>
● Descripción: Cada opción dentro de un < select>.

24. Nombre: textarea

● Inicio: < textarea>
● Cierre: < /textarea>
● Descripción: Caja más grande para escribir texto, como comentarios o mensajes.

25. Nombre: table

● Inicio: < table>
● Cierre: < /table>
● Descripción: Crea una tabla (como en Excel) con filas y columnas.

26. Nombre: tr

● Inicio: < tr>
● Cierre: < /tr>
● Descripción: Fila de la tabla.

27. Nombre: td

● Inicio: < td>
● Cierre: < /td>
● Descripción: Celda de datos dentro de una fila (tr).

28. Nombre: th

● Inicio: < th>
● Cierre: < /th>
● Descripción: Celda de encabezado (título) en una tabla.

29. Nombre: thead

● Inicio: < thead>
● Cierre: < /thead>
● Descripción: Parte superior de la tabla, donde están los títulos.

30. Nombre: tbody

● Inicio: < tbody>
● Cierre: < /tbody>
● Descripción: Parte central de la tabla, donde están los datos.

31. Nombre: tfoot

● Inicio: < tfoot>
● Cierre: < /tfoot>
● Descripción: Parte final de la tabla. A veces se usa para totales.

32. Nombre: nav

● Inicio: < nav>
● Cierre: < /nav>
● Descripción: Sección de navegación. Suele contener enlaces como "Inicio", "Contacto", etc.

33. Nombre: header

● Inicio: < header>
● Cierre: < /header>
● Descripción: Encabezado de la página o sección. Ahí va el logo, título y menú.

34. Nombre: footer

● Inicio: < footer>
● Cierre: < /footer>
● Descripción: Pie de página. Generalmente tiene información extra como derechos de autor o links.

35. Nombre: section

● Inicio: < section>
● Cierre: < /section>
● Descripción: Divide el contenido en secciones con sentido, como capítulos.

36. Nombre: article

● Inicio: < article>
● Cierre: < /article>
● Descripción: Contenido que se puede leer por separado, como un blog o noticia.

37. Nombre: aside

● Inicio: < aside>
● Cierre: < /aside>
● Descripción: Contenido secundario. Ej: una barra lateral o datos relacionados.

38. Nombre: main

● Inicio: < main>
● Cierre: < /main>
● Descripción: La parte principal del contenido de la página.

39. Nombre: meta

● Inicio: < meta>
● Cierre: No tiene cierre
● Descripción: Agrega datos sobre la página (como descripción o codificación). Va en el < head>.

40. Nombre: link

● Inicio: < link>
● Cierre: No tiene cierre
● Descripción: Sirve para enlazar archivos externos, como el CSS.

41. Nombre: script

● Inicio: < script>
● Cierre: < /script>
● Descripción: Inserta código JavaScript que da funcionalidad a la página.

42. Nombre: style

● Inicio: < style>
● Cierre: < /style>
● Descripción: Escribe CSS directamente en el HTML.

43. Nombre: iframe

● Inicio: < iframe>
● Cierre: < /iframe>
● Descripción: Muestra otra página dentro de la tuya, como un video incrustado.

44. Nombre: canvas

● Inicio: < canvas>
● Cierre: < /canvas>
● Descripción: Zona para dibujar cosas con JavaScript (gráficos, animaciones, etc.).

45. Nombre: audio

● Inicio: < audio>
● Cierre: < /audio>
● Descripción: Inserta sonidos o música. Podés darle controles.

46. Nombre: video

● Inicio: < video>
● Cierre: < /video>
● Descripción: Inserta videos, con posibilidad de controles.

47. Nombre: source

● Inicio: < source>
● Cierre: No tiene cierre
● Descripción: Se usa dentro de < audio> o < video> para indicar la fuente.

48. Nombre: code

● Inicio: < code>
● Cierre: < /code>
● Descripción: Muestra código tal cual se escribe.

49. Nombre: pre

● Inicio: < pre>
● Cierre: < /pre>
● Descripción: Muestra texto tal como se escribió, con saltos de línea y espacios.

50. Nombre: mark

● Inicio: < mark>
● Cierre: < /mark>
● Descripción: Resalta el texto como si lo marcaras con un fosforescente
